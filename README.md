### 1. Project Title
Buttercup Restaurant Booking System

---

### 2. Project Description
This project is developed for the IMS566 individual assignment. It showcases a functional web application interface with login authentication, dashboard visualisation, responsive design, and content management features. The project uses HTML, CSS, JavaScript, and a modern front-end framework to demonstrate advanced web design principles.

---

### 3. Features Implemented
- Login page with simulated authentication  
- Dashboard with summary cards  
- Data visualisation using Chart.js   
- Responsive navigation menu  
- Two data view pages  
- Responsive layout (desktop & mobile)  
- Footer and structured content
